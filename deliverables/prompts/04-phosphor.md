# PHOSPHOR — CRT Poetry Zine

## Full Reproduction Prompt

Build a one-page poetry zine that lives inside a 1982 green-phosphor CRT terminal, rendered through a real WebGL tube. Palette: pure black #000000; phosphor green #33FF66 with dimmer tiers #1A8C3C and #0E4A20; warning amber #FFB000. Fonts: IBM Plex Mono for prose, VT323 for display and terminal headings. First build the page as a working DOM terminal: a typed boot sequence overlay (BIOS banner, counting 65,536-byte memory test, device checks, a "[WARN] UNSTABLE" dream module, skippable on any key or click), three melancholic machine-dream poems that type themselves on scroll over invisible ghost copies with blinking block cursors and punctuation-aware delays, an ASCII masthead and dividers, file-metadata headers, an amber system notice, and a working reader console (help / poems / open n / about / dream / clear, arrow-key history, amber "?SYNTAX ERROR" replies). Signature technique — render that live DOM through a raw-WebGL CRT pipeline: mirror it each frame onto an offscreen 2D canvas by walking text nodes and measuring glyph runs with Range client rects (per-character fallback for soft-wrapped lines), drawing borders, block cursors and the input caret yourself; upload it as a texture and post-process with (1) a ping-pong phosphor-persistence buffer — each new frame maxed against the decayed previous frame so bright glyphs leave green trails, (2) threshold + separable-blur bloom at quarter resolution, and (3) a composite shader applying barrel curvature with a black rounded bezel beyond the raster, an RGB aperture-grille mask from mod(gl_FragCoord.x, 3.0), rolling scanlines plus a drifting retrace band, radial chromatic aberration, vignette, grain, flicker, occasional horizontal sync wobble, a degauss ripple on load, and a power-on animation that collapses the raster to a single bright line before blooming open. Forward clicks and hover through the inverse barrel map so every link and the console stay usable through the curved glass; add `crt` (bright / curve / scan / off / on knobs) and `degauss` console commands. Cap devicePixelRatio at 2 (smaller buffers and no chroma taps at ≤768px), pause the rAF loop when the tab is hidden or the canvas is offscreen, render a single static frame under prefers-reduced-motion, and fall back to a pure-CSS scanline/vignette/flicker tube whenever WebGL is unavailable.
