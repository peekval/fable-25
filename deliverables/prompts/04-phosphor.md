# PHOSPHOR — CRT Poetry Zine

## Full Reproduction Prompt

Build a one-page poetry zine that lives inside a 1982 green-phosphor CRT terminal. Palette: pure black #000000; phosphor green #33FF66 with dimmer tiers #1A8C3C and #0E4A20; warning amber #FFB000. Fonts: IBM Plex Mono for prose, VT323 for display and terminal headings. Signature technique — build the CRT entirely from fixed, pointer-transparent overlay layers: repeating-linear-gradient scanlines blended with multiply; a radial-plus-edge vignette; a flicker layer whose opacity jumps through a steps(1) keyframe timeline; a slow radial phosphor breathing glow; and a reusable layered green text-shadow glow token on all bright text, with an amber variant for warnings. On load run a typed boot sequence overlay — BIOS banner, a counting 65,536-byte memory test, device checks, a "[WARN] UNSTABLE" dream module — character by character with per-line pauses, skippable on any key or click (ignore bare modifier keys), and stacked beneath the CRT layers so scanlines wash over the boot too. Type each poem on scroll: keep an invisible ghost copy of the text for stable layout and type into an absolutely positioned live copy with a blinking block cursor and punctuation-aware delays (longer after commas, periods, dashes, line breaks). Finish with a working reader console: prompt input, commands help / poems / open n / about / dream / clear, arrow-key history, amber "?SYNTAX ERROR" replies. Sections: ASCII-art masthead with issue line, three melancholic machine-dream poems with file-metadata headers and ASCII dividers, an amber system notice, terminal footer. Copy: lowercase machine longing against uppercase system chrome. Reduced-motion and no-JS users skip the boot and see poems fully printed.
