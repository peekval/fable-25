# 24 — WOBBLE · Gummy Candy Brand

## Full Reproduction Prompt

Build a single-file HTML site for "WOBBLE", a premium gummy-candy brand that takes joy very seriously. Palette: cream #FFF6E9, bubblegum #FF9EC7, pistachio #A8E6CF, grape #B39DDB, cherry #FF6B81, plum ink #4A2B47 for text and dark sections, soft white #FFFDF9. Fonts: Baloo 2 (600–800) for every headline and button, Quicksand for body. Signature technique: squash-and-stretch physics done entirely in CSS with a springy cubic-bezier(.34,1.86,.44,1) — candy "blobs" are plain divs with layered radial gradients (bright gloss kiss top-left, darker inner shading) whose asymmetric border-radius corners and rotation morph through a 5.5-second wobble keyframe loop, each with a synced squashing ground-shadow pseudo-element; on hover, anything jelly runs a squash keyframe scaling X up while Y compresses (1.28,.72 → .84,1.18 → settle). Hero headline letters drop from -110vh one at a time, overshoot, squash on landing and jiggle into place, each tinted a different brand colour. Wave dividers between sections animate via the CSS d:path() property; a mouse-parallax layer of floating candies shifts by per-element data-depth. Include a dark "flavor lab" console where four gummy-gradient buttons burst 14 randomized confetti dots (CSS custom properties carry vector, size, rotation), increment a wobble counter with milestone microcopy at 10/25/50 presses, and print a deadpan status line. Sections: hero, four flavor cards with jiggle ratings, three-card texture-promise grid, flavor lab, find-us cards, footer with wavy-underline links. Copy tone: playful deadpan-legal humor — "Zero sad chews." Buttons read as gummies via stacked inset shadows. Reduced motion: globally kill all animations and transitions and show everything settled.
