# 21 — DEEP RELAY · Interstellar Mission Control

## Full Reproduction Prompt

Build a single-file HTML mission-control dashboard for "DEEP RELAY", a fictional interstellar probe 103 AU out. Palette: near-black #0A0C10 background with panel gradient #0D1016, phosphor amber #FFB000, signal green #3DFF8C, alert red #FF3B30, grid-line blue-grey #2A3442, body ink #C8D2DC, dim #5C6B7A. Fonts: Share Tech Mono for everything, Rajdhani 700 for the giant hero wordmark. Signature technique: a CRT boot sequence — a full-screen overlay types four green terminal lines (steps() typing animation on width), fades out at ~1.7s, then every panel wipes in with a staggered clip-path inset reveal left-to-right; overlay the whole page with repeating 1px scanlines plus a radial vignette, and lay a faint 48px CSS grid on the body. Give panels amber corner brackets (four absolutely-positioned 10px L-shapes). Fill them with living instruments: a JS clock ticking real UTC and mission-elapsed days; telemetry numbers that random-walk within tight bounds; a circular radar scope whose sweep is a rotating conic-gradient with blips fading in sync; a canvas carrier waveform (amber sine over a green ghost trace); an SVG trajectory plot where a dot travels the flight path via getPointAtLength; a mission log that types entries character by character; and a 60ms inverted "glitch" filter flash every 7–15 seconds. Sections: topbar, hero with spinning SVG reticle, telemetry grid, radar, trajectory, comms, log, instrument-and-crew roster tables, footer. Copy tone: terse NASA ops jargon, all-caps labels, tabular numerals, plausible physics. Honor prefers-reduced-motion by skipping the boot, sweeps and loops; throttle the master requestAnimationFrame loop to ~8fps for data updates.
