# NOCTURNE — Saison 2027 (FABLE/25 · 51)

## Full Reproduction Prompt

Build "NOCTURNE — Saison 2027", a fictional Berlin concert hall's season announcement, as one self-contained HTML file (Google Fonts allowed; one local image, assets/hall.webp). Palette: hall darkness #100A0D, burgundy velvet #5C1E2E, gold #C9A25E with bright #EFD9A7, cream program #F2EAD8, dim #8A7A80. Type: Bodoni Moda as a dramatic didone display with italic accents, Hanken Grotesk for text. All copy in German, Konzerthaus register.

Combine three signature systems. First, stage-light choreography: volumetric beams built as blurred (16px) clip-path wedges of gold linear-gradient with mix-blend-mode screen, one or two per section, each starting rotated ±40–50° at opacity 0; an IntersectionObserver adds a .lit class that transitions them over ~3s to per-section settle angles and colors set via CSS custom properties, so every concert takes the stage lit differently, with short black interlude bands between acts as scene changes. Add a followspot — a fixed 56vmax screen-blended radial gradient that lerps toward the pointer (fine pointers only). Second, music made visible: a fixed full-viewport canvas draws a single elegant score line — a sum of three sines with an edge-taper envelope, stroked as a wide faint halo plus a thin bright core in "lighter" compositing, with ~10 glowing particles riding the curve. Each frame, ease its amplitude toward a target derived from the distance of [data-swell] elements (act titles, hero) to the viewport centre via gaussian falloff, so the page breathes near concerts; add scroll-velocity tremolo as a high-frequency term. Third, the program as five acts: Akt I–V playbill cards (Mahler II, Winterreise, an organ night, Le Sacre, Beethoven IX) with date, program table (italic didone composers, dim work titles), artists, and a soft velvet curtain wipe — two layered burgundy gradients translating apart when lit. After the acts, a seat-selection fan: seven dotted SVG arcs over a stage arc, each row a focusable group with an invisible 26px hit stroke; hover/focus/Enter lights the row bright gold and updates an italic caption (row, price, one-line character).

Generate the hall photo first: prompt Nano Banana 2 (via the Higgsfield MCP) for a 16:9 view from the stage of an 1899 hall at night — four gold-lit balcony ribbons, organ centred on high, a single spotlit conductor's chair, palette-exact — then ffmpeg it to a ~200 KB 1920-wide WebP. Present it full-bleed as "Das Haus" with two overlay beams aligned to the architecture (one fanning down from the organ, one coning up from the stage spot), dark scrims, playbill caption; lazy-load with real dimensions.

Finish with three gold-line Abonnement cards (Parkett/Loge/Première, inner hairline frames, one flagged Meistgewählt) and a Karten section plus footer. Requirements: prefers-reduced-motion lights everything statically (no sweeps, followspot removed, score line drawn once); rAF stops when the tab hides; DPR capped at 2; fewer particles and narrower beams under 720px; zero console errors from file://; no horizontal overflow at 390px.
