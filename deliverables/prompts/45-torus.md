# TORUS — A star, held

## Full Reproduction Prompt

Build a one-page site for TORUS, a fictional fusion-energy company ("A star, held"), as a single self-contained HTML file (Google Fonts allowed; local assets in assets/: plasma.webp still, plasma.mp4 loop, plasma-poster.webp). Aesthetic: futuristic dark — void black #070A0F, steel #2A3140, text #E8ECF2, dim #7A8494, and a plasma gradient from magenta #E24FBB through violet #8E4FE2 to a white-orange core #FFB37A. Fonts: Unbounded for monumental display headlines, Martian Mono for everything else. Copy: awed but engineering-sober.

The showpiece is a scroll-scrubbed ignition sequence. Hero and ignition share one ~560vh section with a sticky full-viewport 2D canvas (no libraries) rendering a plasma torus: ~900 additive-blended particle streaks (340 on mobile) orbiting a squashed elliptical ring via toroidal + poloidal angles, fourteen thin poloidal field-line ellipses hugging the tube, a faint steel silhouette, and a radial core halo. Scroll progress drives everything as a pure function: hero text fades out, a mono HUD types status lines in character-by-character (VACUUM OK → TF COILS AT FIELD 12.6 T → INJECTING D-T → HEATING 40 keV → IGNITION), intensity ramps so particles brighten and shift magenta→violet→white-orange, field lines tighten, and at ~85% a Q-factor readout crosses exactly 1.0 (flipping to orange — net gain) while a radial shockglow overlay washes the viewport with a Gaussian bump that settles to an afterglow plateau, then Q runs away to 11.3. Because everything derives from progress, scrolling back reverses the startup. Pause the rAF loop when the stage is offscreen or the tab hidden; cap DPR at 2.

Generate the assets first: prompt Nano Banana 2 for a 16:9 tokamak interior mid-shot — swirling magenta-violet plasma ring, palette-exact — then feed the still to Kling 3.0 Turbo image-to-video via the Higgsfield MCP for a slow plasma churn, and finish with ffmpeg: a 1920-wide WebP still plus matching poster, and an h264 CRF encode with crossfaded head/tail so the ~4.5s clip self-loops seamlessly around 430KB.

Sections after the stage: "Inside the vessel" — the loop framed as live camera CAM-04 (bordered porthole, corner ticks, blinking REC dot, shot-metadata bar), the video injected by JS only when motion is allowed (autoplay muted loop playsinline preload=metadata, poster) and paused offscreen by an IntersectionObserver. "The machine" — four bordered spec tiles with IntersectionObserver count-up counters (12.6 T field, 150M °C "10× hotter than the sun's core", Q 11.3, 400 s pulse) plus the full-frame still. Roadmap — four-milestone timeline, the 2029 ignition node glowing warm. Footer CTA, fixed "← 25" pill, footer guide link.
