# 25 — SEED · Generative Gallery, Edition of One

## Full Reproduction Prompt

Build a single-file HTML generative-art gallery called "SEED — an exhibition, edition of one". Gallery aesthetic: warm wall #F5F5F2, ink #1C1C1C with softer alpha tints for secondary text, hairline rules everywhere. Fonts: Libre Caslon Text (serif body, italic artwork titles) with Space Grotesk for small tracked-out uppercase labels. Signature technique: a seeded PRNG (mulberry32, seeded from arrival time) drives every decision, so reloading with the same seed would reproduce each work brushstroke for brushstroke. Render four canvas artworks with zero images and zero libraries: (1) a flow field — ~2000 particles stroking short low-alpha segments along sin/cos vector angles, respawning inside a clean composition margin; (2) recursive rectangle subdivision filled as soft-edged Rothko-like colour fields using ~14 jittered low-alpha passes per block; (3) concentric rings whose radii wobble with three random sine harmonics — hand-unsteady by design; (4) ~300 bodies given tangential starting velocity around hidden gravity attractors, tracing coiled orbital threads. Choose one of five curated palettes per visit (e.g. Ink & Vermillion, Dusk, Terracotta & Teal, Moss & Ochre, Plum & Ash), generate titles from shuffled adjective+noun lists, and animate each render over ~70 requestAnimationFrame batches when scrolled into view. Hang the works in white frames with hover lift, radial spotlight glows above, and museum labels: roman numeral, italic title, algorithm name, medium line, pixel dimensions. Add an "Acquire" button that springs in a classic red sold-dot and writes a seed-numbered certificate; a Regenerate button (plus the R key) that fades the wall, retires the seed and mints a new show; and a colophon counting "decisions spent". Copy tone: dry, honest wall text — "arithmetic wearing good light". Reduced motion: render instantly, no lifts or fades.
