# VOXELWERK — small worlds, big hearts

## Full Reproduction Prompt

Build a single-file HTML showcase website for "VOXELWERK", a fictional indie game studio that makes cozy voxel games. Palette: warm clay #E07856, sand #F2E3CE, mint #7FD8BE, deep umber #2A1E17. Fonts: Chivo (900 weight for huge uppercase display type) with Rubik Mono One reserved for tiny labels and numbers. Signature technique: a three.js hero diorama of roughly 2000 voxel cubes rendered as ONE InstancedMesh (single draw call), procedurally laid out in code as a floating island with layered terrain, a small cottage, and trees. On load the cubes assemble cinematically: each instance flies in from a scattered position with staggered delays (bottom layers first, rippling outward from the center) using a gentle back-ease and a decaying tumble. After assembly, stop writing instance matrices. Scrolling orbits the camera smoothly around the island (lerped target angle plus slow constant drift); the cursor tilts the island subtly. Fake the shadow with a blurred radial-gradient plane, never shadow maps. Cap device pixel ratio at 2, pause rendering when the tab is hidden or the canvas is offscreen, use about 800 cubes under 768px, and honor prefers-reduced-motion with a pre-assembled, slowly rotating island. Make the island feel alive: a ~30-second day/night cycle driven only by light-color lerps and a multiply-blend CSS sky tint (never per-instance writes), cottage windows that glow via two additive sprites at dusk, three drifting voxel clouds as tiny once-written InstancedMeshes moved as whole groups, and a handful of additive firefly points that appear only at night. Clicking the island gives a gentle spring-damped scale bounce and a small leaf-particle puff. Sections: sticky full-screen hero with oversized headline over the canvas, a studio manifesto with stat blocks, two game teaser cards using CSS box-shadow pixel art, a full-width third-game announcement capsule ("Wanderlicht — coming 2027") presenting one painted 16:9 key art image (lazy-loaded, real dimensions) revealed by a growing clip-path circle wipe with a few twinkling CSS firefly dots and a warm glow, a second palette-swapped dusk diorama beside a three-step process list, a careers section with hover-reactive role rows, and a chunky footer. Tone: confident, warm, playful indie-studio copy — "cozy is a genre", no lorem ipsum. Style: flat bold blocks, 3px borders, hard offset box-shadows, exactly one image on the whole page (the key art) — everything else is code.
