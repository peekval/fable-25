# THE MERIDIAN — Art-Deco Grand Hotel (Site 08)

## Full Reproduction Prompt

Build a single-file HTML site for The Meridian, a fictional art-deco grand hotel established 1928, restored 2024. Palette: deep emeralds #0B3D2E / #072A20 / #051F17, gold #D4AF6A with pale highlight #F2DFAE, cream #F4EDDF, and a burgundy bar section #6E1E2B. Centerpiece: a raw-WebGL "lobby light" behind the hero type — three depth layers of diagonal volumetric shafts and thousands of gold dust motes that glitter only inside the beams. Render it on a transparent canvas composited additively (premultiplied alpha, blendFunc ONE/ONE, fragments emitting alpha 0) between the CSS hero background and the type: a fullscreen pass sums Gaussian band profiles across the beam-perpendicular axis (a shared GLSL shaftMask function) per layer, each multiplied by fbm density streaming slowly along the beam axis, plus a faint deco sunburst of cosine-power rays deep in the scene that parallaxes slower than the dust, and an elliptical Gaussian guard that dims the light behind the title so type stays crisp. A second pass draws ~2400 CPU-simulated dust motes (900 at ≤768px) as depth-of-field point sprites — nearer motes larger and softer — whose brightness is masked by the same shaftMask, so they catch light only in the shafts; the pointer swirls nearby motes with a decaying tangential + radial impulse so they resettle, and scroll eases the camera for parallax. Cap devicePixelRatio at 2, pause the rAF loop when the hero leaves the viewport or the tab hides, and fall back to static CSS gradient shafts when WebGL is unavailable, the context is lost, or prefers-reduced-motion is set. Fonts (Google): Poiret One for all display type — wide letter-spacing with matching text-indent so centered headings stay optically centered — and Cormorant Garamond for body, italics for asides. Signature one: gold text rendered as a background-clipped linear gradient (bronze → gold → pale gold → bronze) that slowly sheens across headlines. Signature two: self-drawing art-deco SVG ornaments — a hero sunburst of rays, concentric arcs, stepped chevron shoulders and a keystone diamond, every stroke using pathLength=1 with stroke-dasharray/offset and a per-stroke CSS-variable delay so the emblem draws itself in sequence when scrolled into view, fills fading in last. Signature three: a fixed brass elevator panel on the right — round floor buttons (L, M, 5, 41, D) that glow gold when active, italic tooltips, smooth-scroll travel, and scroll-spy via a −42%/−52% rootMargin observer band. Layer a subtle animated feTurbulence film grain and slow parallax conic-gradient ray patterns behind sections. Sections: lobby hero, heritage timeline 1928–2024 on a central gold rail with diamond gems, four room cards with masked 1px gold hairline borders that shimmer on hover, The Gilded Hour bar menu with dotted leaders and prices, a reservations form with underline inputs, footer with a drawn spire mark. Copy tone: wry, impeccable 1928 concierge English. Disable all motion under prefers-reduced-motion.
