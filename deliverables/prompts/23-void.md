# 23 — VOID ATELIER · Monochrome Architecture Practice

## Full Reproduction Prompt

Build a single-file HTML site for "VOID ATELIER", a radically minimalist Basel architecture practice. Strict monochrome palette: #FFFFFF, #F4F4F4, #D9D9D9, #8A8A8A, #3A3A3A, #0D0D0D — no other colour ever. Fonts: Inter at weights 200–400 for enormous ultra-light type, Instrument Serif italic for quiet "whisper" lines. Centerpiece: a raw WebGL fragment shader fixed fullscreen behind the hero (no library) that raymarches ABSENCE — one matte concrete monolith standing in white fog, with everything interesting subtracted from it via smooth SDF boolean subtraction: a sphere scooped from its front face, a doorway cut clean through from the base, and a stair notch descending from its top edge; the subject is the hole, not the object. Shade it with a single raking directional light, five-tap ambient occlusion in the cavities, a ~26-step soft-shadow march, barely-there exponential fog to a paper-white gradient background, and a subtle film-grain pass — high-key, matte, no gloss, no colour. The camera sways almost imperceptibly; the pointer re-aims the light so shadows crawl through the voids; scroll morphs the carved composition (sphere shrinks, doorway widens, stair emerges) while the manifesto sections — each given a solid white or black background — slide over the fixed canvas (z-index:-1). Performance: render at reduced internal resolution upscaled by CSS, cap devicePixelRatio at 2, use fewer march steps and skip the shadow march at ≤768px, pause the rAF loop when the hero leaves the viewport (IntersectionObserver) or the tab hides, and draw the first frame synchronously; prefers-reduced-motion gets a single static rendered frame; if WebGL or shader compilation fails, the canvas stays invisible and the plain page stands. Signature technique elsewhere: reveals behave like light entering a room — elements start fully hidden under clip-path:inset() from one edge (or inset(45% 45% 45% 45%) for images) and ease open to inset(0) over ~1.3s with cubic-bezier(.16,1,.3,1), staggered by .12s, triggered once via IntersectionObserver; hairline 1px rules grow from scaleX(0). Second set piece: a "light study" — a dark concrete room corner built purely from CSS gradients (two walls, seam, floor) crossed by a soft rotating light beam pivoting from an unseen window over 36 seconds while a tabular-nums clock counts 06:00→18:00 in sync. Add custom SVG data-URI cursors (thin crosshair default, surveyor's circle on links), a press-G architect's six-column grid overlay in mix-blend-mode:difference, a 12-second barely-perceptible "breathe" scale on the hero, and project list rows where huge pale ghost numerals fade in behind the title on hover. Sections: full-height hero with two-row uppercase wordmark (second row stroked outline, indented) over the raymarched monolith, practice manifesto in giant 200-weight statements with a serif-italic accent word, black light-study section, three selected works as hairline-ruled rows, centered correspondence block, minimal footer with numbered nav. Copy tone: austere, aphoristic, almost liturgical — "We do not add." Reduced motion: freeze the beam mid-afternoon with the clock matched, disable all reveals; include a noscript fallback that unclips everything.
