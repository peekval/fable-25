# 23 — VOID ATELIER · Monochrome Architecture Practice

## Full Reproduction Prompt

Build a single-file HTML site for "VOID ATELIER", a radically minimalist Basel architecture practice. Strict monochrome palette: #FFFFFF, #F4F4F4, #D9D9D9, #8A8A8A, #3A3A3A, #0D0D0D — no other colour ever. Fonts: Inter at weights 200–400 for enormous ultra-light type, Instrument Serif italic for quiet "whisper" lines. Signature technique: reveals behave like light entering a room — elements start fully hidden under clip-path:inset() from one edge (or inset(45% 45% 45% 45%) for images) and ease open to inset(0) over ~1.3s with cubic-bezier(.16,1,.3,1), staggered by .12s, triggered once via IntersectionObserver; hairline 1px rules grow from scaleX(0). Centerpiece: a "light study" — a dark concrete room corner built purely from CSS gradients (two walls, seam, floor) crossed by a soft rotating light beam (a blurred, clip-path-trimmed gradient strip pivoting from an unseen window over 36 seconds) while a tabular-nums clock counts 06:00→18:00 in sync with the sweep. Add custom SVG data-URI cursors (thin crosshair default, surveyor's circle on links), a press-G architect's six-column grid overlay in mix-blend-mode:difference, a 12-second barely-perceptible "breathe" scale on the hero, and project list rows where huge pale ghost numerals fade in behind the title on hover. Sections: full-height hero with two-row uppercase wordmark (second row stroked outline, indented), practice manifesto in giant 200-weight statements with a serif-italic accent word, black light-study section, three selected works as hairline-ruled rows, centered correspondence block, minimal footer with numbered nav. Copy tone: austere, aphoristic, almost liturgical — "We do not add." Reduced motion: freeze the beam mid-afternoon with the clock matched, disable all reveals; include a noscript fallback that unclips everything.
