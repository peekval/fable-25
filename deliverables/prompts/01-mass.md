# MASS — Kinetic Type Foundry

## Full Reproduction Prompt

Build a single-page site for "MASS", a fictional Swiss type foundry with a loud brutalist attitude. Palette: near-black ink #0A0A0A, acid lime #D8FF3E, warm off-white bone #F2F2EF. Fonts from Google Fonts: Archivo Black for all display type, Space Grotesk for body and UI. Showpiece — a canvas slice-shear wordmark: after document.fonts resolves, pre-render the giant stacked hero wordmark (three rows of MASS: solid bone, stroke-only, acid) to two offscreen canvases — one in true colours, one entirely acid — then overlay a visible canvas on the hero and each frame redraw the buffers as ~44 horizontal slices. Drive a scalar S with an underdamped spring (velocity += (target − S) × 0.085, damped ×0.86) whose target is smoothed scroll velocity ×2.2 clamped to ±120; each slice's x-offset = S×0.55×sin(slicePos×6.5 + phase) + S×0.9×(slicePos − 0.35), with a slight scaleY stretch from |S| — a hard scroll whips the type apart like sheet metal and it settles elastically. Above |S|≈6, composite the acid buffer additively ("lighter") at ±1.9×/0.8× offsets for chromatic echoes. Keep the DOM wordmark as fallback and hide it only after the canvas activates. Second move: two counter-rotating marquee bands (22s/30s loops) skew with velocity, and above a velocity threshold fracture — wrap every glyph in an inline-block span and give each a sine-staggered translateY/rotate scaled by (|velocity| − 16), clearing transforms once settled. Third move: an interactive glyph specimen grid where hover detonates neighbours — cells within a 2-tile radius get a spring impulse away from the epicentre, magnitude (2.6 − distance) × 9, integrated per frame with underdamped springs so tiles shove outward and wobble back. Add a 120px mix-blend-mode: difference cursor blob lerping toward the pointer, scaling ~2.2× over interactive elements. Layout: full-bleed dark canvas, faint 80px grid, vertical-rl labels, clamp() type to 24vw; sections: hero, double marquee, manifesto, hover-invert typeface catalogue rows, glyph grid, three license tiers, acid footer with viewport-wide wordmark. Copy tone: cocky, terse, industrial — "Set it big or don't set it." Idle-guard every subsystem (no draws or style writes once springs settle), gate canvas/fracture work by IntersectionObserver visibility, reset velocity on visibilitychange, and under prefers-reduced-motion skip all of it — never create the canvas, keep the static DOM wordmark.
