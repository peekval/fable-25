# ORBITA — Instruments of Time

## Full Reproduction Prompt

Build a one-page site for "ORBITA", a fictional haute-horlogerie manufacture. Palette: midnight navy #0A0E1A (panel tint #0D1222), silver #C8CDD6 (dim #8A93A5), muted gold #B8934A used at 30% and 15% alpha for hairlines. Fonts: Marcellus for headings and display, IBM Plex Mono with tabular numerals for every number, label and table cell. Signature technique — a Three.js hero: a gyroscope of three nested chrome and gold torus rings around a chrome torus-knot core, each rotating on different axes at different slow rates, the inner ring starting perpendicular for a true gimbal stance. Because polished metal only reads with reflections, bake a procedural PMREM environment from an off-screen scene containing three emissive softbox planes (warm key strip, cool fill, hard top slit) — no image assets — and render with ACES filmic tone mapping, physical lights and navy fog. Dolly the camera back and orbit slightly as the user scrolls through the first viewport (lerped). Pause the render loop when the tab is hidden or the hero scrolls offscreen; cap devicePixelRatio at 2. Add a second set piece: an SVG technical drawing of a gear train whose teeth are generated by script, wheels genuinely meshing (centre distances equal the summed radii), rotating via CSS with per-wheel pivot origins, plus an oscillating balance wheel and pallet fork, mono annotations and a "FIG." plate. Sections: fullscreen hero, craft philosophy with a bordered 2×3 stat grid and blockquote, the movement figure with a numbered feature list, a specification table panel with a softly pulsing "certified" lamp, a 49/49 limited-edition centrepiece with mailto CTA, hairline footer. Tone: austere, reverent, exact. Under prefers-reduced-motion render a single still frame.
