# BABEL — The Infinite Library

## Full Reproduction Prompt

Build a single-file experience site called "BABEL — The Infinite Library", inspired by Borges. Near-black warm palette (paper #EFE6D2, lamp gold #D9A554, wood dark #0E0B08), monospace + classical serif pairing. The centerpiece is a raw-WebGL first-person corridor of hexagonal library galleries: build a few corridor segments (shelves as textured planes, railing, a hanging lamp per gallery, a void above and below) and recycle them seamlessly as scroll walks the camera forward, so the passage is infinite. Fill shelves with procedural book spines rendered to canvas textures: seeded palettes, heights, and titles generated from a syllable grammar. Maintain a gallery counter (huge numbers plus a generated designation like "ra nousvruxvrol"). Every so often one book glows softly; clicking it opens an overlay page of seeded pseudo-language text with exactly one legible English sentence about infinity hidden mid-page. Add "The catalogue": a search field where ANY input is hashed (32-bit) to a deterministic address — gallery, shelf, position — rendered as a card-catalogue entry with a "walk there" button that fast-travels the corridor with acceleration and settle. Include a short Borgesian manifesto section and a colophon. Requirements: the library drifts forward slowly when idle; an FPS watchdog degrades to render-on-scroll on weak rasterizers; IntersectionObserver and visibilitychange pause the loop; prefers-reduced-motion serves a static gallery with the catalogue still functional; no WebGL falls back to a generative 2D shelf wall; no horizontal overflow at 390px; zero console errors. Copy sparse and reverent: "All books. All of them."
