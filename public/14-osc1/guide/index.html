OSC‑1 — Boutique Analog Synthesizer

How it was built

The whole page is one HTML file styled like a piece of studio hardware: a brushed-metal front panel with corner screws, engraved labels, toggle switches and a blinking voice LED, set on a near-black machined background with a faint scanline texture. Michroma carries the uppercase wordmark and headings, IBM Plex Mono everything else.

The centerpiece is a genuinely playable synth voice: the Web Audio API runs four detuned oscillators through a lowpass filter into an analyser, with exponential gain ramps so start and stop never click. Detune weights are deliberately asymmetric so low settings don't collapse into phase cancellation.

Two canvases act as the instrument's oscilloscope and spectrum display — live when audio plays, and fed by a simulated waveform derived from the current knob positions when it doesn't, so the panel always looks powered.

The knobs are SVG dials generated in script, draggable with pointer capture, logarithmic on the cutoff like a real filter pot, and fully keyboard-operable as ARIA sliders.

Performance care is everywhere: scope grids are pre-rendered offscreen, the animation loop pauses when canvases scroll out of view, the trace glow is a cheap double stroke instead of shadowBlur, and reduced-motion visitors get a single static frame.

Generated assets: the two photographs — the three-quarter hero product shot and the FREQ/FINE knob macro — were made with Nano Banana 2 (Higgsfield MCP, text-to-image, 2K), chosen for how precisely it holds studio-hardware materials and lighting. Both prompts were palette-locked by naming the site's hex codes (#17181C body, #FF6B35 cables, #7CFC9A scope glow) so the shots land in-gamut, then optimized to WebP with ffmpeg and framed with a CSS scanline-and-vignette overlay to match the panel chrome.
