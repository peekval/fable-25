HOW DEEP RELAY WAS BUILT

A mission-control console with no images, no libraries — just CSS, one canvas and a small rAF loop pretending to be the Deep Space Network.

How it was builtPNL-01 / DOC

The page opens with a fake boot sequence: four terminal lines type themselves in using a steps() width animation, then the overlay fades and every panel wipes in with a staggered clip-path reveal. A fixed overlay of repeating 1px scanlines plus a radial vignette sits over everything, selling the CRT look, while a faint 48px grid is painted straight onto the body background.

Each panel gets its amber corner brackets injected by a few lines of JS, and the instruments are genuinely alive: a real UTC clock, telemetry values that random-walk inside tight bounds, a radar sweep made from a rotating conic-gradient with blips that fade in sync, and a canvas waveform redrawn at a throttled ~8fps. The trajectory plot is plain SVG — a dot rides the flight path using getPointAtLength — and the mission log types itself entry by entry. A 60ms inverted "glitch" flash every few seconds and a prefers-reduced-motion path that skips all of it complete the illusion.

The showpiece is PNL-07, the downlink optical-nav frame: when the panel scrolls into view, the probe photograph reconstructs like a real deep-space transmission — 28 horizontal scanline segments revealed top-down via a CSS clip-path inset driven by a --rx custom property, an amber phosphor edge line riding the reveal boundary, occasional red "CRC FAIL — RETRANSMIT REQ" flickers, and a mono byte-count readout that ends in a verified checksum line plus a typed mission-log entry. Two more escalations keep the room busy: a full-viewport 300ms amber phosphor sweep every 20 seconds (with a momentary SNR lock-on), and randomized PWS plasma-burst spikes on the carrier canvas that each log a diegetic ops note. Reduced-motion users get the fully rendered frame and no sweeps.

Generated assetsPNL-02 / GEN

The single photograph — assets/probe.webp, the probe's parabolic dish, gold foil and magnetometer boom tiny against the black starfield — was generated with Nano Banana 2 via the Higgsfield MCP at 2K, with the prompt palette-locked to the site's exact colors (#0A0C10 space, #FFB000 amber glint, #3DFF8C accents), then converted and compressed to WebP with ffmpeg. It ships at 1920×1072 with real width/height attributes and lazy loading.
