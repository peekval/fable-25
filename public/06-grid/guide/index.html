How this
was built

SEC. AThe system

GRID° is one HTML file that practices what it preaches: every element — masthead, offset hero lines, essay index, colophon — is placed on a literal 12-column CSS grid with 24px gutters, and a fixed overlay of twelve translucent red columns can be toggled with a button or the G key to prove it. The hero stacks three uppercase Inter Tight headline lines that each begin on a different grid track, with one word rendered as a transparent stroke outline. The essay rows invert on hover using a ::before ink bar that scales up from zero height and bleeds the full viewport width with a negative-inset trick, flipping the type to paper white. A dark manifesto marquee loops seamlessly because JavaScript clones its slogans with aria-hidden, so screen readers hear each line once. Scroll reveals arrive through a single IntersectionObserver that assigns a 70ms stagger to elements entering in the same batch, and everything collapses to a four-column grid under 900px. The only colors on the page are paper, ink, and one red — applied, as the colophon says, surgically.

SEC. BGenerated assets

The page carries exactly one photograph: assets/issue07.webp (1600×1074, 3∶2) — issue №07 finally physical, the printed journal lying open on a white surface with rigorous greeked columns and a single red rule. It was generated with Nano Banana 2 (via the Higgsfield MCP, text-to-image at 2K) from a palette-locked prompt — paper #FAFAF8, ink #111111, one red #E63329, nothing else — then compressed to WebP with ffmpeg.

Its presentation is the point. The plate is mounted on the site's own 12-column grid inside CSS-drawn crop marks and SVG registration marks, and it does not fade in: it assembles from its own grid cells. The photo is sliced into a 6×4 matrix of tiles — each a plain element showing one cell of the image via background-size:600% 400% and a per-tile background-position — which snap on in reading order at 34ms per cell, a red-outlined "hot" cell marking the print head, ending on a rotated "passed for press" stamp. The semantic content stays an ordinary lazy-loaded <img> with real dimensions; under prefers-reduced-motion (or without JavaScript) the tiles never exist and the plate is simply there. Downstream, a 340vh sticky study hard-snaps the grid through ÷1 ÷2 ÷3 ÷4 ÷6 ÷12 divisions as you scroll, and the grid toggle now shows full construction: numbered columns, viewport crop marks, and a spec readout.

Editorial product photograph, top-down slightly angled: a printed Swiss design journal ("GRID°, №07") lying open on a seamless warm-white surface (#FAFAF8). The spread is a rigorous Müller-Brockmann 12-column layout — greeked body text in tight justified columns, one oversized sans-serif headline, small folios — printed in near-black ink (#111111) with exactly one vermilion-red vertical rule (#E63329).
