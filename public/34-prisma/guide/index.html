PRISMA

How the living kaleidoscope was built

The technique

The hero is a single raw WebGL fragment shader — no libraries. It performs a polar fold: every pixel's angle around the centre is wrapped into one wedge and mirrored, so whatever pattern lives in that wedge is repeated N times around the ring, and N itself is a smooth float that morphs between 6, 8 and 12. The source pattern inside the wedge is a stack of drifting sin bands and fractal-brownian-motion noise fields, colour-graded into the four jewel tones over a near-black velvet floor, with citrine deliberately rationed so it reads as gold veins rather than a wash. The mouse is wired straight into the fold: cursor angle rotates the mirror axis, cursor distance from centre drives the zoom, so the audience literally plays the mandala. The three installation thumbnails are the same compiled shader on tiny 120-pixel canvases, all fed by one shared requestAnimationFrame loop that gates every canvas by viewport visibility and pauses entirely on hidden tabs. Under reduced-motion the shader renders one curated still frame and the symmetry buttons step instantly instead of morphing.

Reproduce it

Build a one-page festival site called PRISMA — a light-art and projection festival — as a single self-contained HTML file with a raw WebGL GLSL kaleidoscope hero, no libraries and no images. Palette: black velvet #0A090D background with jewel accents amethyst #9B4DCA, emerald #1FA97A, citrine #F5C542, sapphire #2E6BD6. Google Fonts: Monoton solely for the neon-deco hero word "PRISMA" (gradient-filled text with a slow animated hue-rotate), Outfit for everything else. Hero: full-viewport fragment shader that polar-folds the plane into N mirrored wedges (atan/mod/abs), where N is a float smoothly morphing between 6, 8 and 12 via pill buttons and a slow auto cycle; the wedge is filled with layered moving sine bands and fbm noise fields graded into the jewel palette over deep velvet shadows, citrine kept sparse as golden veins. Interaction: mouse angle rotates the fold axis, mouse distance from centre controls zoom, eased with frame-rate-independent smoothing and shortest-path angle interpolation. Sections: about with stat grid, three installation cards each showing a live 120px mini kaleidoscope (same shader, DPR 1, shared rAF loop, IntersectionObserver visibility gating, max three concurrent), an artists lineup grid, three ticket-night cards, footer. Cap hero DPR at 1.25 for 60fps. Honour prefers-reduced-motion with a frozen still frame and instant control steps. Copy should be electric but curated — a nocturnal turbine hall in Zürich, no lorem ipsum.
