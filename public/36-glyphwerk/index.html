01

The renderer, not the framework

Frameworks expire. Projection does not. At Glyphwerk you build the machinery yourself — point by point, character by character — until nothing on your screen is a mystery to you.

Most creative-coding education starts at the top: import a library, call a function, admire the result. We start at the bottom. Week one, your entire toolkit is a grid of characters and a loop.

This is not nostalgia. Constraint is a teaching instrument. A 110-column text grid forces you to understand sampling, aspect ratio, luminance and depth — because every mistake is legible, literally.

By the time you touch a GPU, you will have already written one in miniature. That order of operations changes what kind of programmer you become.

Draw before you importEvery technique is first built from primitives you wrote yourself. Dependencies are earned, never assumed.
Text is a displayA character grid is a framebuffer with opinions. Learn to shade with a density ramp and pixels become trivial.
Ship the frame loopSixteen milliseconds is the deadline that teaches honesty. Every studio project must hold 60fps on a five-year-old laptop.
02

Ten lines, one artwork

lesson-00.js· waiting

A ten-line JavaScript loop that places nine hundred glyphs on a golden-angle spiral. The code types itself while the canvas beside it executes the loop; afterwards two constants sweep continuously and the artwork follows them.

Output — running this exact loop

A spiral of monospace glyphs growing as the code is typed, then breathing as two constants oscillate.

The code writes itself · the canvas obeys 900 glyphs · phyllotaxis

This is the whole pedagogy in one exhibit. The snippet is real JavaScript, not pseudocode — the canvas on the right is executing precisely that loop as it is typed. When the typing ends, watch the two highlighted constants begin to sweep: change a number, change the artwork. Fourteen weeks of Glyphwerk is that feedback loop, taken seriously.

Type as Material

Letterforms treated as substance: kinetic typography, text that flows along fields, glyph grids that carry images. You will learn why a monospace advance width is a unit of measurement, and how to make language physically move.
