HOW MERCVRY WAS BUILT

01 / THE READHow it was built

The liquid-metal hero is a 2D metaball field: six blobs whose r²/d² influence is summed per pixel at quarter resolution into an offscreen buffer, then upscaled — pixels above the threshold get a vertical chrome palette, a blue-white rim just above the isoline, darkened cores, and an electric #4D5BFF halo on the anti-aliased skirt. One blob chases the cursor; on touch it drifts autonomously.

The headlines are chrome text: a ten-stop silver gradient clipped into the glyphs with background-clip:text, its background-position eased toward the cursor's X so the "environment" slides across the metal — wrapped in an @supports guard so unsupported browsers fall back to solid silver. Magnetic buttons follow the pointer and release with a springy overshoot, the marquee blurs proportionally to scroll velocity, and the specimen drawings animate their gradients with SMIL — which the script strips under reduced motion, since CSS media queries can't stop SMIL. The whole simulation pauses via IntersectionObserver whenever the hero leaves the viewport.
