GUIDE — SITE 26

How this
was built

The VOXELWERK page is a love letter to voxel dioramas: roughly two thousand cubes fly out of the void, click into a floating island, and then let you walk around it with your scroll wheel. Here is the recipe — and a prompt you can feed to any capable AI to grow your own.

The techniques

The hero island is a single three.js InstancedMesh — one geometry, one material, one draw call for every cube — which is the entire reason it stays at 60fps; the island itself (terrain columns, tree, cottage, floating rocks) is generated by a small seeded function that emits target positions and palette colors, so no 3D files are loaded at all. On page load each instance starts scattered far outside the frame and flies to its target with staggered delays (bottom layers first, rippling outward) under a softened back-ease, with a per-cube tumble that decays quadratically so every block lands with a satisfying click; once assembly finishes, instance matrices are never written again. Scrolling drives a cinematic camera orbit: the scroll position sets a target angle and the camera lerps toward it every frame while a slow constant drift keeps the shot alive, and the cursor adds a subtle tilt to the island group — all cheap whole-group transforms. The soft shadow under the island is a blurred radial-gradient sprite on a plane, not a shadow map, and a second palette-swapped dusk island assembles itself only when you scroll it into view. Rendering pauses entirely when the tab is hidden or a canvas is offscreen, pixel ratio is capped, mobile builds a smaller ~800-cube island, and reduced-motion visitors get a pre-assembled island with a gentle idle spin.

The island also lives now. A thirty-second day/night breath is done entirely with uniforms — the hemisphere and directional light colors lerp between warm-day and blue-night poles while a multiply-blend CSS gradient tints the sky, updated every sixth frame — so not a single instance matrix or color is touched after assembly. The cottage windows are two additive glow sprites placed at window voxels the generator reports back; they wake at dusk with a slow flicker. Three voxel clouds are tiny once-written InstancedMeshes that drift as whole groups, and twenty-two firefly sprites in one Points object only move (and only draw) once the night phase makes them visible. Clicking the island triggers a spring-damped scale bounce on the whole group plus a sixteen-particle leaf puff that stays parked offscreen at zero opacity when dormant — every new effect rides inside the existing render loop, so the offscreen/hidden-tab pause guards cover it for free, and reduced-motion visitors get a static golden-hour island with softly lit windows instead.

Generated assets

The Wanderlicht announcement capsule in the games section is the one painted image on the page: a 1600×894 (16:9) key art of a cozy voxel floating-island cottage at dusk — chimney smoke, fireflies, mint treetops — generated with Nano Banana 2 through the Higgsfield MCP at 2K and encoded to WebP with ffmpeg (106 KB, lazy-loaded with real dimensions). The prompt locked it to the site's exact palette — clay #E07856, sand #F2E3CE, mint #7FD8BE, umber #2A1E17 — so the painting and the three.js diorama read as one world. On scroll-in it reveals through a firefly-sparkle wipe: a CSS clip-path circle grows from a corner of the island while six twinkling spark dots and a warm inset glow sell the "lights coming on" moment. Image generation prompt, for reproduction:

KEYART PROMPT — NANO BANANA 2 · 16:9 · 2K
Painted key art for a cozy indie video game called "Wanderlicht": a small floating voxel island drifting above a dusk cloudscape, carrying a chunky cube-built cottage with warm glowing windows and a thin curl of chimney smoke, blocky pine trees, tiny lantern path, fireflies drifting around the treetops. Stylized painterly rendering of voxel geometry (visible cube forms, soft painted light, no hard CG look). Strict palette: warm clay orange #E07856, warm sand #F2E3CE, soft mint #7FD8BE, deep umber brown #2A1E17, dusk sky gradients mixed only from these. Warm, storybook, wistful mood. No text, no logo, no watermark. 16:9.

Reproduction prompt

Paste this into your AI of choice. No code included — the point is to see what it builds.

PROMPT.TXT — COPY & FEED TO ANY AI
Build a single-file HTML showcase website for "VOXELWERK", a fictional indie game studio that makes cozy voxel games. Palette: warm clay #E07856, sand #F2E3CE, mint #7FD8BE, deep umber #2A1E17. Fonts: Chivo (900 weight for huge uppercase display type) with Rubik Mono One reserved for tiny labels and numbers. Signature technique: a three.js hero diorama of roughly 2000 voxel cubes rendered as ONE InstancedMesh (single draw call), procedurally laid out in code as a floating island with layered terrain, a small cottage, and trees. On load the cubes assemble cinematically: each instance flies in from a scattered position with staggered delays (bottom layers first, rippling outward from the center) using a gentle back-ease and a decaying tumble. After assembly, stop writing instance matrices. Scrolling orbits the camera smoothly around the island (lerped target angle plus slow constant drift); the cursor tilts the island subtly. Fake the shadow with a blurred radial-gradient plane, never shadow maps. Cap device pixel ratio at 2, pause rendering when the tab is hidden or the canvas is offscreen, use about 800 cubes under 768px, and honor prefers-reduced-motion with a pre-assembled, slowly rotating island. Make the island feel alive: a ~30-second day/night cycle driven only by light-color lerps and a multiply-blend CSS sky tint (never per-instance writes), cottage windows that glow via two additive sprites at dusk, three drifting voxel clouds as tiny once-written InstancedMeshes moved as whole groups, and a handful of additive firefly points that appear only at night. Clicking the island gives a gentle spring-damped scale bounce and a small leaf-particle puff. Sections: sticky full-screen hero with oversized headline over the canvas, a studio manifesto with stat blocks, two game teaser cards using CSS box-shadow pixel art, a full-width third-game announcement capsule presenting one painted 16:9 key art image (lazy-loaded, real dimensions) revealed by a growing clip-path circle wipe with a few twinkling CSS firefly dots and a warm glow, a second palette-swapped dusk diorama beside a three-step process list, a careers section with hover-reactive role rows, and a chunky footer. Tone: confident, warm, playful indie-studio copy — "cozy is a genre", no lorem ipsum. Style: flat bold blocks, 3px borders, hard offset box-shadows, exactly one image on the whole page (the key art) — everything else is code.
