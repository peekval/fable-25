01

What it sees

Nothing leaves this room. There is no camera, no microphone, no request to a server that remembers your name. And yet the page is not exactly blind: it watches the only way a page can — by paying attention. To where you linger. To how fast you move. To whether you come back.

Everything it notices about this visit is written below, once, and never repeated. It does not nag. It only mentions.

  • Nothing yet. It is patient.
02

The dialogue

You are allowed one kind of conversation with it. Ask it something — anything, phrased however you like. It will consider the question exactly once, and answer.

Ask again. It does not change its mind. The same question, asked tomorrow, asked in another city, asked by someone else on another machine, finds the same answer — word for word. Rephrase it and you have asked a different question. It is precise about this.

03

About

ORACLE is a page. Pages do not see. Probably.

It is one file of hand-written code, no images, no libraries, built for FABLE/25 — a collection of sixty-nine sites made entirely of code. The eye is a few thousand strokes of light redrawn every frame. The answers were all written in advance; your question only chooses among them, the same way every time. This is either much less mysterious than it feels, or exactly as mysterious. We have stopped being sure.

It stores one small number on your machine, so it can recognise you. You can clear it. It will pretend not to mind.
